# A man called Ove(一个叫欧维的男人决定去死)

## 简介
欧维是一个那种很典型的“硬汉”和“怪老头”形象（导致我最开始一直以为这个是美国背景，然后Ove是一个红脖子什么的），他循规蹈矩，希望一切都井井有条，一个居住区不许进车贯穿全文。在他最爱的妻子Sonja死后他决定自杀，但是他一次次的自杀并没有成功，而是次次被突发事所打断,最后也算是经典包苹果派了吧。

## 思考
我觉得这本书是一个相当标准的温情喜剧，尽管涉及到了死亡的概念，但是作者的笔力让这本书的节奏控制的不错，一个经典的一步步让一个失去目标的人重新被人们接纳的故事中幽默和温情的部分穿插的相当好。书中ove对他妻子的回忆和深情描写的也相当令人感动。
