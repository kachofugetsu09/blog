# 《地下室手记》

## 简介

这个书是一个地下室人的自述，他是那种高傲知耻，但是沾点自甘堕落并且有些享受这种堕落带来的羞耻的人。他清楚的认知到自己是一个有病的，无耻的人，他同时为之感到痛苦。整个书分为上下两部分，第一部分是他的自述和哲学思考（日记？）向看官叙述里他自己的理论。第二部分是他回想往事，写他和他的同学之间的关系以及主体是和妓女丽莎相识的故事。

## 思考
其实写书的这个有一点局外人的感觉（描写了一个人，一个社会承认之外的人），但是这个人的病态相当严重，文字之间像是换了俄罗斯特产热病（也有可能是牢陀特色，罪与罚也有点这个感觉，但是没这个严重）。一个高傲病态而阴暗潮湿的人，反向多余人了。
