# 《索拉里斯星》

## 简介

心理学家克里斯·凯尔文奉命前往环绕神秘行星“索拉里斯”的空间站，调查科研人员精神异常与接连死亡的原因。索拉里斯表面被一整个具有意识的海洋覆盖，它似乎能读取人类潜意识，并将人类内心最深的记忆与罪疚“具象化”为真实存在。
随着凯尔文亲身经历这些不可思议的现象，他逐渐意识到，人类并非在研究外星智慧，而是在被迫直面自身的内心与局限。
## 思考
这本书构建出的原生质海洋生命体和那种人类中心的太空歌剧和外星人接触不一样，这本书的文明接触更富有异质性，是一个思维形态或者整体的形态就跟人类完全不同的生命体索拉里斯之海和人类之间的关系。这里索拉里斯之海无意识的，机械般的模拟出基于人潜意识的生命更是风格经典但是实现独特的科幻人文思考。记忆，投影什么的，很酷。
怪不得鸣潮和舟都用了索拉里斯设定，确实酷。
