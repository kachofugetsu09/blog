# 《树上的男爵》

## 简介
柯西莫在12岁时因为拒绝吃蜗牛而跑到了树上，从此一生生活在树上再也没有下来过。他在树上的一生很是精彩，名扬天下海外。他在树上读书，结交朋友，参与劳动，恋爱，写作度过了一个充实的人生。他甚至见了拿破仑，见到了一族被限制在树上生活的贵族，当他们选择在惩罚结束后下地后他继续选择留在了树上。

## 思考
一个生活在树上的人应当是淡漠超然的隐者。但是写一个在树上生活但是依旧热爱着生活的人，一个热爱着人与人之间的联系的人则反而更有力度。在树上他是自由的，卡尔维诺的说法完美的概括了这个书的思想：“一个人只有远离人群，才能真正和他们在一起”。    

ps：这本书的语言很美很轻，一贯的卡尔维诺的风格
